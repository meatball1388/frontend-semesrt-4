# Контрольные работы по дисциплине "Фронтенд и бэкенд разработка"

Этот репозиторий содержит материалы контрольных работ №1 и №2.

## Структура репозитория

### 1. [frontend-1cr](./frontend-1cr)
**Контрольная работа №1: Основы фронтенда и бэкенда**
- Практические занятия (1-5)
- Базовые компоненты React, работа со стилями (SCSS)
- Простейшая серверная часть на Node.js

### 2. [frontend-2cr](./frontend-2cr)
**Контрольная работа №2: Полноценное веб-приложение**
Полноценное веб-приложение для управления товарами с системой аутентификации и разграничением прав доступа (RBAC).

#### Основные возможности:
*   **Аутентификация:** Регистрация и вход с использованием JWT (Access + Refresh токены).
*   **Ролевая модель (RBAC):** `guest`, `user`, `seller`, `admin`.
*   **Безопасность:** Хеширование паролей (`bcrypt`), обновление токенов через интерцепторы `axios`.
*   **Документация:** Swagger API документация.

#### Структура:
*   `/server`: Node.js + Express сервер.
*   `/client`: React.js приложение.

---

### Запуск проекта (на примере CR №2):

**Сервер:**
1. `cd frontend-2cr/server`
2. `npm install`
3. `node server.js`

**Клиент:**
1. `cd frontend-2cr/client`
2. `npm install`
3. `npm start`
